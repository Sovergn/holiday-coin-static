HolidayCoin is BitCoin for Charity

HolidayCoin is a cryptocurrency similar to Bitcoin, except that all proceeds from the token sale(1) are donated to charity. Holders of the token get to vote on which charity receives the funds.

Token Sale Dec 25-31, 2017

The HolidayCoin project is a one-time sale, during the holiday season 2017. Token sale starts Dec 25 and lasts until midnight New Years Eve. HolidayCoin tokns cost $1 each(2).
